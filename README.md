# Brain-pitch

An interactive, single-file explainer for [Brain](https://github.com/Pappydapimp69/Brain) —
the git-backed shared knowledge system four linked repos (`memory`, `ideas`,
`tension`, and your project) use to carry lessons across AI coding sessions.

Open `index.html` directly, or serve it statically (e.g. GitHub Pages). No
build step, no dependencies — plain HTML/CSS/JS with the display and console
typefaces inlined as data URIs.

Every command output and kernel quoted in the page is real, pulled from
actual commits in `pappydapimp69/memory`, `pappydapimp69/ideas`, and
`pappydapimp69/tension` — nothing is dramatized for the pitch.
